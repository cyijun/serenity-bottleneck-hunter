## 一句话主题定位

A股机器人产业链的资本开支确定性来自**Tesla Optimus量产、国内宇树/智元/优必选等本体厂扩产、以及工业自动化升级**；钱必须花在减速器、电机、丝杠、轴承、力/视觉传感器等核心部件上，周期可看3-5年。本期聚焦**上游材料 + 中游核心器件**的冷门瓶颈股，下游整机巨头仅作参照。

## 关键瓶颈层识别

1. **L3 中游核心器件（减速器 / 丝杠 / 力传感器）**：技术壁垒高、验证周期长、国产替代空间大的环节同时命中原型 **② 单一来源卡脖子** + **④ BOM-普适**。谐波减速器、行星滚柱丝杠、六维力传感器是当前人形机器人量产最紧的“卡脖子”点。
2. **L5 上游材料（稀土永磁 / 软磁 / 非晶）**：电机BOM必需、中国具备资源优势、市场关注度远低于器件层，命中原型 **① 上游材料垄断** + **④ BOM-普适** + **⑦ 冷门/前机构**。

## 最大主题级风险 / Bear Case

1. **人形机器人量产进度不及预期**：Tesla Optimus、国内本体厂的量产时间表多次推迟，导致核心零部件订单放量慢于当前估值隐含假设。
2. **估值已大幅前置**：绿的谐波、雷赛智能、埃斯顿等核心标的3个月涨幅>100%，部分已进入抛物线；一旦主题退潮，高估值小票首当其冲。
3. **国产替代节奏慢于预期**：日本HD、Nabtesco在高端减速器仍具技术与成本优势，若降价反击或扩产，国产厂商份额与毛利率承压。
4. **下游本体厂压价**：机器人整机竞争激烈，成本压力向上游传导，导致“有量无价”，核心器件毛利扩张不及预期。
